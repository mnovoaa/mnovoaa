### Sobre mí

Construyo productos web de extremo a extremo e integro IA donde aporta de verdad. Fundé
**[ShowUp](https://showup.lat)**, un estudio de sitios web a medida para artistas y marcas, y mi
proyecto de título **[TailFind](https://github.com/mnovoaa/tailfind)** une visión por computador,
CLIP y LLMs en una PWA real.

- 🧠 Me interesa la arquitectura de sistemas, la integración de LLMs y el diseño de producto.
- 🛠️ Trabajo principalmente con **Next.js / React / TypeScript** y **.NET / Blazor**.
- 🔐 Base en ciberseguridad — Cisco **Ethical Hacker**.
- 🌎 Español nativo · inglés intermedio.
